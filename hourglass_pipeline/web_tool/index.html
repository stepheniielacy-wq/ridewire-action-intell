How a claim gets verified

  1. The Learner gathers a claim with citations from a real source (forum thread, manufacturer doc, repair video).
  2. The Debater checks how many independent domains — not just repeated links from the same site — back the claim.
  3. 3+ independent domains → status VERIFIED. Fewer → NEEDS_MORE_SOURCES, and it stays pending, visible, and unused until more corroboration arrives.
  4. Only VERIFIED claims get formatted into the Pro and DIY reports above.
